# Project 2 - Exploratory Data Analysis (EDA)

## Overview

This project focuses on performing Exploratory Data Analysis (EDA) on an e-commerce dataset to identify trends, patterns, and outliers. The analysis helps understand customer behavior, product demand, payment preferences, and sales performance.

## Objectives

* Perform descriptive statistical analysis
* Identify sales trends and business patterns
* Analyze product demand
* Evaluate payment method preferences
* Analyze referral sources
* Detect outliers in transaction values

## Tools Used

* Python
* Pandas
* Matplotlib
* Jupyter Notebook

## Dataset Information

* Total Records: 1200
* Dataset Type: E-commerce Orders Dataset

## Analysis Performed

### 1. Dataset Overview

* Checked dataset shape and structure
* Reviewed available columns and records

### 2. Descriptive Statistics

* Calculated Mean
* Calculated Median
* Calculated Count
* Analyzed Minimum and Maximum values

### 3. Product Analysis

* Identified most purchased products
* Compared product order frequencies

### 4. Payment Method Analysis

* Analyzed customer payment preferences

### 5. Order Status Analysis

* Evaluated order distribution across different statuses

### 6. Referral Source Analysis

* Identified major customer acquisition channels

### 7. Monthly Sales Trend Analysis

* Analyzed sales performance over time
* Identified highest and lowest sales periods

### 8. Outlier Detection

* Applied the IQR (Interquartile Range) method
* Identified unusual transaction values

## Key Findings

* Average Order Value: ₹1053.97
* Median Order Value: ₹823.62
* Most Purchased Product: Printer
* Most Preferred Payment Method: Online
* Top Referral Source: Instagram
* Highest Sales Month: June 2024
* Outliers Identified: 8

## Conclusion

The Exploratory Data Analysis provided valuable insights into customer purchasing behavior, sales trends, and business performance. The analysis identified key patterns in product demand, payment preferences, referral sources, and monthly sales trends. Outlier detection helped highlight unusual transactions for further investigation.
